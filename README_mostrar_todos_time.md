# Página com botão “Mostrar todos do time”

Suba estes arquivos na raiz do repositório:

- `index.html`
- `style.css`
- `script.js`
- `estadios.js`

## Nova função

Ao escolher um time no filtro **Time/Equipo**, aparece o botão:

- PT: `Mostrar todos do time`
- ES: `Mostrar todos del equipo`

Quando ativo, ele mostra todos os jogos disponíveis daquele time no JSON, ignorando:

- data
- próximos 3 dias
- próximos 7 dias
- cidade
- região

Ele mantém o filtro de campeonato, caso você queira ver todos os jogos do time dentro de um campeonato específico.
